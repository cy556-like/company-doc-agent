常见问题

关于 DocAgent，你可能想知道的

DocAgent 支持哪些文档格式？

DocAgent 支持 PDF、Word（DOCX）、Excel（XLSX）、TXT、Markdown 等主流文档格式。上传后系统自动完成解析、分段与语义索引，无需手动配置。

数据安全如何保障？

全链路 TLS 1.3 加密传输，支持企业内网私有化部署，数据不离开组织边界。基于 RBAC 的细粒度权限控制，全量操作留痕可审计，满足等保与 GDPR 要求。

部署需要多长时间？

SaaS 版本即开即用，5 分钟即可完成账号创建与知识库初始化。私有化部署通常在 1-3 个工作日内完成，包含环境配置与基础数据导入。

是否支持自定义 Agent 能力？

企业版支持通过 API 集成自定义工具与插件，可以扩展 Agent 的检索、分析和执行能力，满足行业特定场景需求。

免费版和企业版有什么区别？

免费版提供 3 个用户账号和 100MB 知识库空间，适合小团队试用评估。企业版不限用户与空间，支持私有化部署、Agent 自定义和 SLA 保障。
